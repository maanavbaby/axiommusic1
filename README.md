# 🎵 Axiom Music Bot

A powerful Telegram music bot with smooth streaming, premium UI, fast response and advanced controls.

## ✨ Features

- 🎶 Play Music from YouTube
- 📜 Queue System
- ⏭ Skip / Pause / Resume
- 🔁 Loop Support
- 🔍 Song Search
- 📥 Download Support
- 🖼 Auto Thumbnail Generator
- 👥 Group Music Playback
- ⚡ Fast & Stable Performance

## 🛠 Environment Variables

| Variable | Description |
|---|---|
| API_ID | Telegram API ID |
| API_HASH | Telegram API HASH |
| BOT_TOKEN | Telegram Bot Token |
| STRING_SESSION | Pyrogram Session |
| MONGO_DB_URI | Mongo Database URL |
| OWNER_ID | Bot Owner ID |
| LOG_GROUP_ID | Log Group ID |

---

## 💻 Local Deployment

```bash
git clone https://github.com/maanavbaby/axiommusic1
cd axiommusic1
pip install -r requirements.txt
python Axiom.py
```

---

## 📝 Commands

### Common
- `/start` - Start bot
- `/help` - Help menu
- `/ping` - Check bot speed

### Music
- `/play` - Play song
- `/pause` - Pause music
- `/resume` - Resume music
- `/skip` - Skip track
- `/end` - Stop playback
